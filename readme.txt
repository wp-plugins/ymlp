== Description ==

### Integrate Gravity Forms with YMLP

Gravity Forms is the best form plugin for WordPress. YMLP is one of the best email marketing companies out there. It's a match made in heaven!

#### __Each form submission gets added to YMLP__ as a contact.

It takes a couple of minutes to set up, and from then on, you won't have to worry about exporting your contact form submissions and importing them into your YMLP email group.

If you have questions, comments, or issues with this plugin, <strong>please leave your feedback on the <a href="http://wordpress.org/tags/gravity-forms-ymlp?forum_id=10">Plugin Support Forum</a></strong>.

== Installation == 

1. Upload plugin files to your plugins folder, or install using WordPress' built-in Add New Plugin installer
1. Activate the plugin
1. Go to the plugin settings page (under Forms > Settings > YMLP)
1. Enter the information requested by the plugin.
1. Click Save Settings.
1. If the settings are correct, it will say so.
1. Follow on-screen instructions for integrating with YMLP.

== Frequently Asked Questions == 

= How do I find my YMLP API Key? =
To find your API key, log in to YMLP, then go to the Configuration tab and click on the sub-navigation link "API".

= How do I add fields to the Map Data Fields list? = 
To add more Data Fields, go to Manage Contacts &rarr; Manage Data Fields in YMLP.
